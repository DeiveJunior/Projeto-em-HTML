Módulo 6 – Listas


Listas ordenadas: <ol><li>Item 1</li><li>Item 2</li></ol>

Exemplos:

  1. Item 1
  2. Item 2
  3. Item 3

Listas não ordenadas: <ul><li>Item 1</li><li>Item 2</li></ul>

Exemplos:

  • Item 1
  • Item 2
  • Item 3

Listas alinhadas ( lista dentro de lista )

O atributo type altera a forma como os itens da lista são exibidos. Em listas ordenadas (<ol>), ele é válido e pode exibir números, letras ou algarismos romanos. Já em listas não ordenadas (<ul>), ele existia para alterar o marcador da lista, mas é considerado obsoleto em HTML5, sendo recomendado utilizar CSS para essa alteração.

Exemplo:

  • Item 1
    • Item 1.1
    • Item 1.2
  • Item 2
    • Item 2.1
    • Item 2.2
  • Item 3
    • Item 3.1
    • Item 3.2

Personalização (type="A", type="1")

Exemplo:

  1. Item 1
  2. Item 2
  3. Item 3
  • Item 1
  • Item 2
  • Item 3

Módulo 7 - Tabelas

As tabelas em HTML são utilizadas para organizar dados em linhas e colunas, facilitando a visualização e a comparação das informações. Elas são indicadas para exibir dados tabulares, como listas, horários, notas e preços.


Exemplo de tabela:

Nome Idade Sexo
Juliano 27 Masculino
Guilherme 27 Feminino
Deivi Gostosão sim todo dia bb

Módulo 8 – Formulários

Os formulários em HTML são utilizados para coletar informações do usuário, como nome, e-mail, senha, opções e outros dados, permitindo que essas informações sejam enviadas para processamento.

Exemplo:

Dados Pessoais
